# Screenshoty pro nápovědu

Tento adresář obsahuje screenshoty pro help modal. Potřebné screenshoty:

## Požadované screenshoty:

1. **quick-navigation.png** - Screenshot ikon rychlé navigace
   - Zobrazit tlačítka rychlé navigace pod nadpisem otázky
   - Ukázat ikony: Shrnutí, Studijní text, Audio, Další zdroje, Kartičky, Kvíz
   - Rozměry: doporučeno 800x400px nebo větší

## Jak vytvořit screenshoty:

1. Otevři stránku v prohlížeči
2. Použij nástroj pro screenshot (např. Windows: Win+Shift+S, nebo nástroje prohlížeče)
3. Ulož screenshoty s názvy uvedenými výše
4. Optimalizuj velikost souborů (doporučeno použít PNG nebo WebP)

## Poznámka:

Pokud screenshoty nejsou k dispozici, zobrazí se placeholder text. Screenshoty se načítají automaticky, pokud jsou v tomto adresáři.
